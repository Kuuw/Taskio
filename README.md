## Building

### Prerequisites
-   [.NET SDK](https://dotnet.microsoft.com/download)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Kuuw/Taskio.git
    cd Taskio
    ```

2.  **Restore dependencies**
    ```bash
    dotnet restore
    ```

3.  **Build the solution**
    ```bash
    dotnet build
    ```
